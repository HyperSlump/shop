# hyper$lump

A digital storefront for industrial sound design textures and raw synthesis. Built with Next.js, Stripe, Supabase, and Vercel Blob.

## Features

-   **Stripe Integration**: Secure checkout for digital products.
-   **Automated Delivery**: Instant download links via Vercel Blob.
-   **Supabase Database**: Order tracking and customer management.
-   **Responsive Design**: Mobile-first "Acid Goth" aesthetic.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/hyperslump.git
    cd hyperslump
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up environment variables:
    Copy `.env.local.example` to `.env.local` and fill in your API keys for Stripe, Supabase, and Vercel Blob.

4.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
